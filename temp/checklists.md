**시스템 코어 및 클록 설정 (MCU & Reset)**

- **MCU 코어 클록 설정**: MAX32560 (32-bit RISC ARM Cortex-M3)의 동작 주파수를 **108.48MHz**로 정확히 구성합니다.
- **외부 오실레이터(X-TAL) 활성화**:
    - 시스템 메인 클록을 위한 **12MHz** Crystal 활성화 코드를 구현합니다.
    - RTC 및 저전력 모드를 위한 **32.768kHz** 부하 클록 설정을 확인합니다.
- **리셋 라인 풀업 확인**: 내부 풀업 저항이 50kΩ ~ 150kΩ 범위로 존재하므로, 초기화 시 리셋 핀의 내부 풀업/풀다운 레지스터 설정이 하드웨어와 충돌하지 않는지 검토합니다.
- **디버그 인터페이스 가동**: 개발 초기 단계에서 JTAG(TDI, TMS, TCK, TDO) 포트가 정상 인식되는지 핀 맵을 확인합니다.

**2. 외부 메모리 인터페이스 (QSPI Flash)**

- **QSPI 드라이버 구현**: U2 메모리칩인 **W25Q128JVPIM (128Mb / 16MB)** 사양에 맞춰 Quad SPI 통신 프로토콜을 구현합니다.
- **핀 맵 정합성 검증**: **OSPI_CSN, OSPI_CLK, OSPI_DO, OSPI_DI, OSPI_WPN, OSPI_HDN** 핀들이 MAX32560의 QSPI 전용 기능 핀으로 알맞게 매핑되었는지 GPIO 초기화 코드를 확인합니다.
- **동작 모드 설정**: 대용량 데이터 및 펌웨어 리소스 전송을 위해 시리얼 모드가 아닌 Quad 모드(4-bit I/O)가 정상 활성화되는지 레지스터를 체크합니다.

**3. 보안 및 탬퍼 감지 (Secure Box)**

- **탬퍼 물리 스위치 처리**: 물리 스위치 감지선인 **TAM3, TAM5 Pair**의 인터럽트 설정을 구성합니다.
- **메시 패턴(Mesh Pattern) 감지**: PCB 레이어 상의 물리적 탈착이나 손상을 감지하는 **TAM0, TAM1, TAM2 Pair** 라인의 상태 변화를 실시간 혹은 초저전력 모드에서 모니터링하도록 설정합니다.
- **보안 이벤트 대응**: 탬퍼 감지 시 내부 시크릿 키나 민감 데이터를 즉각 삭제(Zeroization)하는 하드웨어 암호화 블록 연동 코드를 준비합니다.

**4. 통신 및 외부 페리페럴 (NFC, USB, UART)**

- **NFC 컨트롤러 (PN5180)**:
    - SPI 인터페이스(MOSI, MISO, SCK, NSS) 및 제어선(**BUSY, IRQ, NRESET**)의 드라이버를 구현합니다.
    - ⚠️ **[초특급 주의사항] RF 전원 로직 제어**: 도면에 명시된 주석에 따라 `RF_PWR_ON` 신호는 **LOW일 때 ON, HIGH일 때 OFF**로 동작합니다. 단, "PCI 버전 보드에서는 ON/OFF가 반대로 동작"하므로 보드 버전에 따라 하이/로우 출력을 반전시키는 조건부 컴파일 코드가 필수적입니다.
- **USB 2.0**: USB0 인터페이스를 활성화하고, 케이블 삽입 여부를 판단할 **DET (VBUS 감지)** 핀과 **RESET** 핀 인터럽트 루틴을 구성합니다.
- **UART 0/1 Debug**: 개발 로그 출력 및 RS232 통신을 위한 UART 포트를 초기화합니다.

**5. 사용자 인터페이스 (LED & Buzzer)**

- **I2C 기반 LED 드라이버 (LP5012)**:
    - **LED_SDA, LED_SCL**을 통해 I2C 통신을 초기화하고, 칩을 깨우기 위해 **LED_EN** 핀을 HIGH로 출력합니다.
    - 총 4 채널의 RGB LED(총 12개 채널 제어)에 대응하는 PWM 레지스터 제어 플러그인을 작성합니다.
- **부저(Buzzer) 제어**: 트랜지스터 구동 방식이므로, `MCU_BUZ` 핀에 기설정된 PWM 주파수를 인가하여 원하는 비프음 톤을 구현합니다.

**6. 전원 관리 및 배터리 감시 (Power & Battery)**

- **전원 안정성 체크**: 부스트 컨버터(TPS61252DSG)의 **PWR_GOOD** 신호를 모니터링하여 메인 5V 전원이 정상 공급되는지 체크하는 예외 처리 로직을 추가합니다.
- **백업 배터리 감시 (TPS3619-33DGK)**:
    - 배터리 모니터링 전압이 **2.0V 이하**로 떨어질 때 **#PFO (#Power Fail Output, VBAT_OK)** 신호가 **HIGH**로 바뀌는지 확인하는 인터럽트 감지 코드를 구성합니다.
    - 메인 전원 상실 시 MCU가 백업 모드(None-Activation Mode)로 안전하게 진입하여 코인 셀 배터리(CR2032) 수명(약 5~6년)을 보존하도록 저전력 슬립 모드 진입 알고리즘을 설계합니다.
